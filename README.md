# インフルエンザ感染症分析

インフルエンザ感染者数の分析・予測プロジェクト

## プロジェクト構成

```
├── configs/          # 設定ファイル（ハイパーパラメータ等）
├── data/
│   ├── raw/          # 生データ（IDWR週報、気象データ）
│   │   ├── idwr/     # 感染症発生動向調査週報 CSV
│   │   └── weather/  # 都道府県別気象データ CSV
│   ├── processed/    # 前処理済みデータ
│   └── external/     # 外部データソース
├── notebooks/        # Jupyter Notebook（探索・分析用）
├── reports/
│   └── figures/      # 生成されたグラフ・図表
├── src/
│   ├── data/         # データ取得・前処理
│   ├── features/     # 特徴量エンジニアリング
│   ├── models/       # モデル定義・学習
│   ├── utils/        # ユーティリティ
│   └── visualization/ # 可視化
└── tests/            # テストコード
```

## データソース

- **IDWR**: 感染症発生動向調査週報（国立感染症研究所）- 都道府県別週次感染者数
- **気象データ**: 都道府県別の気温・湿度・降水量（週次）
